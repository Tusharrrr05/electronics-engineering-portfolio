ESP32 LED Blink

Objective

Blink an LED using the ESP32 microcontroller.

Components Used

- ESP32
- LED
- Connecting Wires

Working

The ESP32 turns the LED ON for 1 second and then OFF for 1 second continuously, creating a blinking effect.

- led_blink.ino - Source code
