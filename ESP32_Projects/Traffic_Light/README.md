# ESP32 Traffic Light

## Objective
Implement a traffic light system using ESP32 and three LEDs.

## Components
- ESP32
- Red LED
- Yellow LED
- Green LED

## Working
The red LED glows for 5 seconds, yellow for 2 seconds, and green for 5 seconds repeatedly.
