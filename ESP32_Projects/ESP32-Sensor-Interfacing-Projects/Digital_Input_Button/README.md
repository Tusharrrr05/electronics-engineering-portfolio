# Digital Input - Push Button Control

## Objective
To understand digital input reading using ESP32 by controlling an LED with a push button.

## Components Used
- ESP32
- Push Button
- LED
- Resistor

## Working
The ESP32 continuously reads the state of the push button through a GPIO input pin. When the button is pressed, the LED turns ON. When the button is released, the LED turns OFF.
