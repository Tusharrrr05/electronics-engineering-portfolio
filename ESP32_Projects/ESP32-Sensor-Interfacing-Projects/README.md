# ESP32 Sensor Interfacing Projects

## Overview
This repository contains basic ESP32 sensor interfacing and GPIO projects developed using the Wokwi simulator. The purpose of these projects is to understand digital input, digital output, analog input, analog output (PWM), and sensor interfacing concepts using the ESP32 microcontroller.

## Projects Included

### 1. Digital Output - LED Blink
- Blink an LED at a fixed interval using ESP32 GPIO.

### 2. Digital Input - Push Button LED Control
- Read a push button input and control an LED based on button state.

### 3. Analog Input - LDR Sensor Reading
- Read light intensity values from an LDR using the ESP32 ADC.

### 4. Analog Output - PWM LED Brightness Control
- Control LED brightness using PWM signals generated by the ESP32.

### 5. LDR Controlled LED (Automatic Street Light)
- Read light intensity from an LDR.
- Automatically turn ON/OFF an LED depending on ambient light conditions.
