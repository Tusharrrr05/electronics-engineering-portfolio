# Analog Output - PWM LED Brightness Control

## Objective
To understand analog-like output generation using PWM by controlling LED brightness with the ESP32.

## Components Used
- ESP32
- LED
- Resistor

## Working
The ESP32 generates a PWM signal on a GPIO pin. By changing the duty cycle of the PWM signal, the average voltage applied to the LED changes, resulting in different brightness levels ranging from dim to fully bright.
